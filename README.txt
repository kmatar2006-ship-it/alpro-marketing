Alpro 3D Pot Viewer — embed package

Files:
  alpro-pot-3d.html   — the viewer (slim, references images by filename)
  label.jpg             — your label wrap

To embed on your site:
  1. Upload all files in this zip to the same folder on your server.
  2. Use an iframe:
       <iframe src="alpro-pot-3d.html"
         style="width:500px;height:500px;border:none;"></iframe>

Because images are external files (not inlined), the HTML stays small —
~13 KB — easy to paste into chat tools, edit by hand, or feed to coding agents.
